\# 📊 Dashboard Baires



Aplicación desarrollada en Python con Streamlit para la gestión y visualización de ventas.



\## 🚀 Características



\- Registro de ventas

\- Visualización de datos

\- Base de datos local SQLite

\- Interfaz simple e intuitiva



\## 🛠️ Tecnologías utilizadas



\- Python

\- Streamlit

\- SQLite

\- Supabase (integración futura)



\## ▶️ Cómo ejecutar el proyecto



1\. Clonar el repositorio:

git clone https://github.com/brandyovalles60-sys/sistema-Dashboard-.git



2\. Entrar a la carpeta:

cd sistema-Dashboard-



3\. Crear entorno virtual:

python -m venv venv



4\. Activar entorno:

venv\\Scripts\\activate



5\. Instalar dependencias:

pip install -r requirements.txt



6\. Ejecutar aplicación:

streamlit run DashboardBaires.py
